Detailed Description

GUIFactory provides access to various dialogs and control.

Some dialogs and User Controls are constructed after a camera object has been connected. End user can use these dialogs to operate on associated camera object. They can also embed User Controls into own applications.

Dialogs that are available without connecting a camera:

  1. CameraSelectionWindow
  2. PropertyGridWindow
  3. ImageDrawingWindow

User Controls that are available without connecting a camera:

  1. CameraSelectionControl
  2. PropertyGridControl
  3. ImageDrawingControl
